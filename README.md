# Salman Arefin Portfolio

A one-page developer portfolio built with Next.js, React, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Then open: http://localhost:3000

## Customize

- Update your projects in `app/page.jsx`
- Replace `/public/Salman_Arefin_CV.pdf` with your real CV file
- Replace contact links with your exact LinkedIn and GitHub URLs
- Add project screenshots inside `/public` and update the project cards

## Deploy

The easiest option is Vercel:

1. Push this folder to GitHub
2. Go to Vercel
3. Import your GitHub repository
4. Click Deploy
